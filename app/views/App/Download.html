At the moment there are no prebuilt binaries for the Abyss Engine so you will have to build it yourself To build it yourself you will need the go runtime, you can get it here. Once you have the go engine you will need to get the Abyss Source to do that run: 

            
                go get github.com/OpenDiablo2/AbyssEngine
                cd AbyssEngine
Then to get the dependencies run: 
            
                go get -d
Finally to run Abyss Engine: 
            
                go run .
